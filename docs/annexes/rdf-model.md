# RDF model definition and diagrams (Informative)

## Model definition

The SPDX RDF ontology is expressed in RDF/OWL/SHACL format
and is published in online at
[SPDX 3.0.1 Model](https://spdx.org/rdf/3.0.1/spdx-model.ttl)
